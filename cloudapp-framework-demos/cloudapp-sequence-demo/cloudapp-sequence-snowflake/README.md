# cloudapp-sequence-snowflake

no

## Configuration

```yaml
io:
  cloudapp:
    sequence:
      snowflake:
        enabled: true
        workerId: # worker id
        workerIdBits: # worker id bits
        sequenceBits: # sequence bits

```

## Run demo

Open the CloudappSequenceSnowflakeDemoApplication class and run it.
